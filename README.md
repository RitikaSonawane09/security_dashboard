# Security System

## Features
- JWT Authentication
- Risk-based Access Control
- Redis-based Real-time Tracking
- ML Anomaly Detection
- Security Dashboard (Charts + Logs)

## Tech Stack
- Flask
- Redis
- Chart.js
- Python (ML)

## How to Run
1. Install dependencies
2. Run Redis
3. Start Flask app

## Dashboard
Real-time monitoring of:
- Risk scores
- Denied requests
- Anomaly detection
